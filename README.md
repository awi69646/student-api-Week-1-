# Student API with FastAPI

This is a simple Student Management API built using FastAPI and Python. 
It allows you to:

- Get all students
- Get a student by ID
- Add a new student
- Update an existing student
- Delete a student

## Requirements

- Python 3.8+
- FastAPI
- Uvicorn
- Pydantic
